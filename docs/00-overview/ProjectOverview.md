## Purpose

Merchant Harmony is a cloud-native customer engagement platform that enables merchants and customers to communicate through structured feedback conversations initiated by scanning a merchant QR code.

The MVP focuses on fast onboarding, structured feedback, and clean service boundaries.

---

## Core Capabilities

- Merchant registration
- Customer registration
- SMS OTP authentication
- Merchant QR onboarding
- Merchant Landing
- Merchant-Customer association
- Structured feedback conversations

---

## Architecture Summary

- Java 21
- Spring Boot 4.1.0
- Maven multi-module
- auth-service
- engagement-service
- Database per service
- REST APIs
- Docker
- Flyway

---

## Guiding Principles

- Simplicity first
- Excellent onboarding
- Clear ownership
- Independent deployment
- API-first design
